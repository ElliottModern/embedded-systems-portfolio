Title

Project 04 - STM32 External Button Interrupt LED Toggle

Goal

Toggle the onboard LD2 LED using a hardware interrupt generated by the B1 push button.

Hardware

STM32 NUCLEO-F446RE
onboard LD2 LED (PA5)
B1 button (PC13)

Key concepts used

external interrupts
EXTI controller
GPIO interrupt configuration
NVIC interrupt routing
interrupt callback functions

Button characteristics

PC13 input
active LOW
falling edge interrupt

Interrupt path

Button press
→ GPIO PC13
→ EXTI line 13
→ NVIC interrupt
→ EXTI15_10_IRQHandler
→ HAL_GPIO_EXTI_Callback()

Core code

void HAL_GPIO_EXTI_Callback(uint16_t GPIO_Pin)
{
    if (GPIO_Pin == GPIO_PIN_13)
    {
    	LD2_GPIO_Port->ODR ^= LD2_Pin;
    }
}

Main loop

while (1)
{
    // system idle
}

Notes

The firmware is now event-driven instead of time-driven.
The CPU remains idle until a button press triggers an interrupt.
The interrupt handler toggles the LED without polling the button in the main loop.
